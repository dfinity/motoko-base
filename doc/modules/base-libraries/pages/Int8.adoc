[[module.Int8]]
= Int8

8-bit signed integers with checked arithmetic

Most operations are available as built-in operators (e.g. `1 + 1`).

[[value.toInt]]
== toInt

Conversion.

[source,motoko]
----
let toInt : Int8 -> Int
----

[[value.fromInt]]
== fromInt

Conversion. Traps on overflow/underflow.

[source,motoko]
----
let fromInt : Int -> Int8
----
